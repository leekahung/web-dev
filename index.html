About Me

View Resume

Although I've been around word processers and spreadsheets for many years in school, it wasn't until my time as a researcher did I get my first real exposure to programming. In research, there were a number of tasks that could only be done through a computer, so I had to "learn to code" in order to complete them.

Over time, I've managed to create simple code to solve problems, used Bash scripts to "automate the little things" on the terminal, but I was hardly proficient at it. It was in late 2021, I've went and built my very first website from scratch. While it looked silly and still is completely out of wack on mobile, it was surprisingly fun and a very refreshing experience.

It never occured to me before, but rather than just letting the computer compute results in the background like I've always done, it was really cool to just see the components I've made just seconds ago working on the web in real time. It was just phenomenal!

By late August 2022, I was fully engaged in building my current developer portfolio. I'm currently learning to build Full-stack apps using the MERN stack and hosting them on Netlify.
