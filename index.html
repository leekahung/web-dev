Although I've been around computers for many years, it wasn't until my time as a researcher in grad school did I came across the world of coding. As a researcher, I had to perform various tasks that could only be done through a computer, so I taught myself how to code functional programs. First with basic Bash scripts, then with Python.

In 2021, I've came picked up new languages during my time at a Data Science bootcamp, namely HTML, CSS, and JavaScript and built my very first website from scratch. While it looks silly now (since I haven't learn about Responsive Web Design at the time), it was a rather fun and refreshing experience. Rather than just letting the computer or the browser compute results in the background, being able to see the very components I've built in action was phenomenal.

By late August 2022, I was fully engaged to building my current developer portfolio and building pet projects. I'm currently learning to build Full-stack apps with the MERN stack and hosting them on Netlify.
